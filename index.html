Javascript Notes

Functions are commands that tell the computer what to do. In javascript, functions are always followed with "();". The paranthesis are filled with paramters which tell the function what to do specifically. Parameters vary from function to function. For the ellipse function, the parameters are (x,y,w,h) which tell the computer the x position, y position, width and height of the ellipse you want to draw.
ex. ellipse(200, 200, 300, 350);
